# SpringMediCareWeb Selenium Testing

## Tên đề tài
Kiểm thử tự động cho hệ thống quản lý đặt lịch khám bệnh dùng Selenium WebDriver

## Mô tả
Đây là repository phục vụ đồ án kiểm thử tự động cho hệ thống quản lý đặt lịch khám bệnh SpringMediCareWeb.

## Mục tiêu
- Tìm hiểu hệ thống quản lý đặt lịch khám bệnh.
- Thiết kế test case cho các chức năng chính.
- Viết test tự động bằng Selenium WebDriver.
- Ghi nhận lỗi, sửa lỗi và kiểm thử lại.
- Lưu trữ tài liệu, mã nguồn và bằng chứng kiểm thử.

## Công nghệ sử dụng
- Java
- Spring MVC
- Hibernate
- MySQL
- Selenium WebDriver
- JUnit/TestNG

## Cấu trúc thư mục dự kiến
- SpringMediCareWeb: mã nguồn hệ thống đặt lịch khám bệnh.
- docs: tài liệu kiểm thử, test case, bug report.
- evidence: hình ảnh minh chứng lỗi và kết quả kiểm thử.
- selenium-tests: mã nguồn kiểm thử tự động bằng Selenium WebDriver.
